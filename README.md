# EXPIREX PRO – MariMart Edition
Aplikasi Monitoring Expired Date untuk toko retail — versi khusus MariMart.

## Features
- Dashboard expired lengkap
- Input barang (barcode + manual)
- Hybrid Date Picker (scroll & ketik)
- Status Expired otomatis
- Prioritas barang mendekati ED
- Tema orange + biru MariMart
- Siap terhubung backend (Google Apps Script)

## Teknologi
- HTML + CSS + JavaScript
- Netlify Hosting
- Backend via Google Apps Script (API)

## Cara Deploy (Netlify)
1. Upload repo ini ke GitHub
2. Buka Netlify → “Import from Git”
3. Pilih repository
4. Deploy

## Backend API
Akan ditambahkan pada versi berikutnya (Google Apps Script)
